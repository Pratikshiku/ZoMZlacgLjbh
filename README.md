### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 旅游景点
5. 纪念品
6. 特色美食
7. 游记信息
8. 系统公告
9. 后台管理
10. 留言板
11. 购物车
12. 个人中心
13. 我的订单
14. 我的地址
15. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 产品分类管理
4. 用户管理
5. 景点分类管理
6. 旅游景点管理
7. 纪念品管理
8. 特色美食管理
9. 游记信息管理
10. 系统管理
11. 订单管理
